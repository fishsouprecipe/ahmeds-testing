## Installation

```bash
# Pass your github credentials

$ cd ahmeds-testing
$ python3 -m venv venv
$ . venv/bin/activate
(venv) $ python3 -m pip install -r requirements.txt
(venv) $ playwright install chromium
```

```bash
(venv) $ python3 -m app add

# Pass your telegram credentials
```

## Run
```bash
(venv) $ python3 -m app run
```
